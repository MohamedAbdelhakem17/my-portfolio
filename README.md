# 🚀 My Personal Portfolio — M. Abdelhakem

This is my personal portfolio website built with **Next.js (App Router)**.
The goal of this project is to showcase my work, services, resume, and contact details in a clean, fast, and modern experience.

## 🛠️ Tech Stack

- **Next.js 14** (App Router)
- **React**
- **TypeScript**
- **Tailwind CSS**
- **next/font/local** for custom fonts
- **SEO Optimized** (Metadata, Open Graph, sitemap, robots.txt, JSON-LD)
- **Responsive Layout**

---

## 📌 Features

- 🏠 **Home Page** — Hero section, intro & featured work
- 🛠️ **Services Page** — List of services and what I offer
- 📄 **Resume Page** — Experience, education & skills
- 🎨 **Work Page** — Projects & case studies
- ✉️ **Contact Page** — Contact form & social links
- 🔍 **Full SEO Setup**
  - Meta Tags
  - OG Images
  - Sitemap
  - robots.txt
  - Schema.org JSON-LD
